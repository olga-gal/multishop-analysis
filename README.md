MultiShop Business Analysis

Author: Olga Gal

Date: 06.06.2026

Tools: Microsoft Excel (XLOOKUP, SUMIFS, MINIFS, Pivot Tables)

This project presents a business analysis of a fictional e-commerce store "MultiShop". Data was generated randomly using AI tools, then processed and verified in MS Excel.
The goal of the analysis was to examine the profitability of sales channels, product categories, and sales trends based on key business indicators (KPIs).

Analysis Scope: 
sales channel profitability (Allegro, Empik Marketplace, Amazon, Shopify); product category analysis (Outdoor, Cosmetics, Home, Electronics); geographic analysis (6 major Polish cities); monthly sales trends and seasonality

Files:
MultiShop_Analysis.pdf - dashboard and conclusions;
MultiShop_Analysis.xlsx - full Excel workbook with raw data, pivot tables and dashboard
